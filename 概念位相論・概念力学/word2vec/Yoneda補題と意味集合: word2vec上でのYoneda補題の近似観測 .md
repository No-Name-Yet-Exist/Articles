概念位相論において
概念 Cは意味射ベクトルの集合として定義されます
C = { a₁_vec, a₂_vec, ..., aₙ_vec }

Yoneda補題
For any object A in a category C,
and any functor F: C → Set,
Nat(Hom(–, A), F) ≅ F(A)

訳: 対象A = すべての「Aに向かう射」の集合
「対象Aは、Hom(–,A)によって特徴付けられる」

正に意味集合のことです。
